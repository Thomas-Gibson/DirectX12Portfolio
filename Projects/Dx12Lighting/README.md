# Lighting Demo

https://github.com/user-attachments/assets/30986eeb-7350-4c23-a1e5-17d15c7ec017


## Description
This demo shows a Lambertian diffuse lighting calculation to light up a piano model. For this project I wanted to work with a 3D model. I implmeented an AssetLoader class that uses ASSIMP in the back end to load the model. To view the model I also made a fly camera to look around. The video shows the lambertian pixel shader, fly camera, and model loading in action.
